# 2026-04-21 日志

## custom-tab-bar 重建
- 问题：之前误删了 custom-tab-bar 的所有文件，同时 app.json 没有 `"custom": true`，导致导航栏降级为系统原生（只有文字无图标）
- 解决：重建了 index.wxml / index.wxss / index.js / index.json，app.json 加回 `"custom": true`
- 图标用 CSS 纯 div 绘制（不用 SVG 标签，因为小程序不支持内联 SVG）：首页=房子、广场=地球、消息=气泡、我=人像
- 注意：app.json tabBar.list 的四个页面必须都是 tabBar 页面，custom-tab-bar 才能在这些页面上显示

## custom-tab-bar 图标优化
- 改用 base64 内嵌 SVG 的 image 标签（小程序不支持内联 SVG，image+base64 是可靠方案）
- 选中/未选用两套 SVG（颜色不同），用 wx:if 切换
- 添加 :active 缩放点击效果（transform: scale(0.85)）
- 修复 updateSelected 方法：通过 getCurrentPages() 获取当前路由自动匹配选中项
- 消息图标红点用 position:absolute 的 view 叠加，svg-icon 需设 position:relative
